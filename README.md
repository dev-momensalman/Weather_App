## ğŸš€ Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (Stable)
- [Dart SDK](https://dart.dev/get-dart)
- An API Key from [OpenWeatherMap](https://openweathermap.org/api) (or your preferred provider).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/momen-salman/Weather_App.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Weather_App
   ```

3. **Install dependencies:**
   ```bash
   flutter pub get
   ```

4. **Run the app:**
   ```bash
   flutter run
   ```
